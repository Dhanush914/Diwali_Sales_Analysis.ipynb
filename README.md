Diwali Sales Data Analysis

📖 Overview
This project analyzes customer purchasing behavior during the Diwali festival using Python-based data analysis techniques. The objective is to uncover insights that can help businesses improve customer targeting, optimize marketing strategies, and increase sales.

🎯 Objectives
Understand customer demographics and buying patterns
Identify high-performing customer segments
Analyze sales trends across different categories
Provide data-driven business insights

🛠️ Tech Stack
Programming Language: Python
Libraries: Pandas, NumPy, Matplotlib, Seaborn
Environment: Jupyter Notebook

📂 Dataset
Diwali Sales Dataset (CSV file)
Includes customer details such as:
Gender
Age Group
State
Occupation
Product Categories
Purchase Amount

⚙️ Project Workflow
Data Cleaning
Removed null values
Dropped irrelevant columns
Data Transformation
Changed data types
Renamed columns for better readability
Exploratory Data Analysis (EDA)
Gender-wise analysis
Age group analysis
State-wise sales distribution
Occupation-based insights
Data Visualization
Bar charts
Count plots
Distribution plots

* Key Insights
Female customers contributed more to overall sales
The 26–35 age group showed the highest purchasing activity
Married individuals had higher purchasing power
Certain states contributed significantly to total revenue
IT, Healthcare, and Aviation professionals were top buyers
🚀 Results

The analysis provides valuable insights into customer behavior during festive seasons, helping businesses:

Improve targeted marketing
Optimize product offerings
Enhance customer segmentation
